# RoomieFind 🏠

A flatmate finder web app for Mumbai built with React JS and Firebase.

🔗 **Live Demo:** [https://monu655.github.io/roomiefind/](https://monu655.github.io/roomiefind/)

## Features
- Google Login
- Browse flatmate listings
- Post your own listing
- Filter by area, budget, gender, occupation, age
- Save listings
- Connect with potential flatmates
- Data stored in Firebase Realtime Database

## Tech Stack
- React JS
- Firebase Auth (Google Login)
- Firebase Realtime Database
- Vite

## How to Run

```bash
npm install
npm run dev
```

Open http://localhost:5173
